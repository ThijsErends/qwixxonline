# Qwixx Online

A mobile-friendly progressive web app (PWA) for the dice game Qwixx.

## About

This is a digital scorecard for Qwixx, designed to replace the physical score sheets used in the game. The app provides a simple, intuitive interface optimized for mobile devices.

## Features

- Digital scorecard for Qwixx
- Mobile-friendly responsive design
- Works offline as a PWA
